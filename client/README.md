# Frontend Development Guide

- for big component, e.g., modal, wallet, and main content, use
  - `rounded-2xl`: border radius
  - `p-5`
  - `m-5`

- for small component, e.g., wallet selector, use
  - `rounded-lg`
  - `p-2.5`

- for button
  - `py-2.5 px-6`

for title: `text-3xl`
